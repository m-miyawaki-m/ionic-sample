# 画面パターン集 設計仕様書

## 概要

倉庫管理の代表的な 3 画面パターンをモック実装。
外部ライブラリ追加なし、各ファイル完結のベタ書き。
検索は setTimeout + ion-spinner でローディング再現。

## ファイル構成

```
src/views/samples/screen-patterns/
├── ScreenPatternsIndex.vue     ← 一覧ページ
├── ReceivingPattern.vue        ← 入荷検品
├── ShippingPattern.vue         ← 出庫
└── InventoryPattern.vue        ← 在庫照会
```

## 共通レイアウト

```
┌─ Header ──────────────────────────┐
│ ← タイトル              [一括登録] │
├─ 入力エリア ──────────────────────┤
│ ○○番号 [________] [検索]          │
├─ リストエリア ────────────────────┤
│ No│[×]│品番│品名│数量│登録日時│詳細│☑│
│  1│[×]│A001│部品A│ 10│04/09 │ 🔗│☑│
│  2│[×]│A002│部品B│  5│     │ 🔗│☐│
│ (検索中は ion-spinner を中央表示)   │
├─ ボタンエリア ────────────────────┤
│           [登録]  [印刷]          │
└───────────────────────────────────┘
```

## リスト行の構成要素

| 要素 | 説明 |
|------|------|
| No. | 自動採番（1, 2, 3...） |
| 削除ボタン | 行先頭の × アイコン。タップで行削除（confirm alert） |
| 品番/品名等 | 検索結果から自動セット |
| 数量等 | 手動入力可能フィールド（type=number） |
| 登録日時 | 一括登録ボタンで現在日時がセットされる |
| テキストリンク | タップでダイアログ表示（詳細情報） |
| チェックボックス | 登録対象を手動で選択 |

## 操作フロー

1. 番号を手入力 → [検索] 押下
2. ion-spinner 表示（0.8秒）
3. 結果がリストに追加（No.自動採番）
4. リスト内の数量等を手動編集
5. チェックボックスで登録対象を選択
6. [一括登録] → チェック済み行の登録日時を現在日時で一括更新 → toast で完了通知
7. テキストリンク押下 → alert で詳細情報表示
8. [×] → confirm alert → 行削除
9. [印刷] → toast「印刷しました」

## 3 画面の差分

| 要素 | 入荷検品 | 出庫 | 在庫照会 |
|------|---------|------|---------|
| 検索対象 | 入荷伝票番号 | 出庫指示番号 | 品番/品名 |
| リスト列 | No/×/品番/品名/予定数/実数/登録日時/詳細/☑ | No/×/品番/品名/指示数/出庫数/登録日時/詳細/☑ | No/品番/品名/ロケ/在庫数/詳細 |
| 手動入力 | 実数 | 出庫数 | なし（照会のみ） |
| チェック | あり | あり | なし |
| 削除ボタン | あり | あり | なし |
| 一括登録 | あり | あり | なし |
| 印刷 | あり | あり | あり |
| 詳細ダイアログ | 入荷履歴 | 出庫履歴 | 在庫推移 |

## データの持ち方

各ファイル内に ref + ダミーデータ配列をベタ書き。外部ファイルなし。

```typescript
const items = ref<RowItem[]>([]);
const loading = ref(false);
let nextNo = 1;

const dummyResults = [
  { code: 'A001', name: 'ボルト M8×30', plannedQty: 10 },
  { code: 'A002', name: 'ナット M8', plannedQty: 5 },
];

const search = () => {
  loading.value = true;
  setTimeout(() => {
    for (const d of dummyResults) {
      items.value.push({ no: nextNo++, ...d, actualQty: 0, checked: false, registeredAt: '' });
    }
    loading.value = false;
  }, 800);
};
```

## ローディング表示

検索中はリストエリアに ion-spinner を中央表示。

```html
<div v-if="loading" class="ion-text-center ion-padding">
  <ion-spinner name="crescent" />
  <p>検索中...</p>
</div>
```
