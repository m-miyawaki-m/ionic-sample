## 1. 在庫管理の基本概念

| 用語 | 英語 | 説明 |
|------|------|------|
| WMS | Warehouse Management System | 倉庫管理システム。入出庫・在庫・ロケーション・ピッキング等を統合的に管理するシステム |
| SKU | Stock Keeping Unit | 在庫管理の最小単位。同じ商品でもサイズや色が異なれば別SKUとなる |
| 在庫 | Stock / Inventory | 倉庫に保管されている商品の総称 |
| 理論在庫（帳簿在庫） | Book Inventory | システム上の在庫数。入出庫の記録から算出される |
| 実在庫 | Physical Inventory | 実際に倉庫にある在庫数。棚卸しで確認する |
| 在庫差異 | Inventory Discrepancy | 理論在庫と実在庫の差。差異がある場合は原因調査が必要 |
| 有効在庫 | Available Inventory | 在庫のうち、引当されていない出荷可能な数量 |
| 引当（ひきあて） | Allocation | 受注や出荷指示に対して在庫を確保すること。引当済み在庫は他の注文に使えない |
| 引当解除 | Deallocation | 引当を取り消し、在庫を有効在庫に戻すこと |
| 安全在庫 | Safety Stock | 需要変動や納品遅延に備えて保持する最低限の在庫量 |
| 発注点 | Reorder Point | 在庫がこの数量を下回ったら発注する基準値 |
| リードタイム | Lead Time | 発注から入荷までにかかる日数 |
| デッドストック | Dead Stock | 長期間動きがなく、今後も出荷見込みのない在庫 |
| 滞留在庫 | Slow-moving Inventory | 一定期間出荷がない在庫。デッドストックの予備軍 |
| 預かり在庫 | Consignment Stock | 他社から預かっている在庫。所有権は預け元にある |

## 2. 入出庫・検品

| 用語 | 英語 | 説明 |
|------|------|------|
| 入荷 | Receiving / Inbound | 商品が倉庫に届くこと |
| 入荷検品 | Receiving Inspection | 入荷した商品が発注通りか確認する作業。品目・数量・品質をチェック |
| 検収 | Acceptance Inspection | 入荷検品に合格し、正式に受け入れること |
| 入庫 | Put-away / Stocking | 検品済みの商品を所定のロケーションに格納する作業 |
| 出庫 | Issue / Retrieval | ロケーションから商品を取り出すこと |
| 出荷 | Shipping / Outbound | 商品を倉庫から送り出すこと |
| 出荷検品 | Shipping Inspection | 出荷する商品が指示通りか照合する作業 |
| 返品 | Return | 出荷済み商品が戻ってくること。返品入荷として処理 |
| 横持ち | Cross-docking | 入荷した商品を在庫にせず、そのまま出荷エリアに移す方式 |
| ASN | Advanced Shipping Notice | 事前出荷通知。入荷前にサプライヤーから届くデータ |

## 3. ロケーション管理

| 用語 | 英語 | 説明 |
|------|------|------|
| ロケーション | Location / Bin | 倉庫内の保管場所。棚番とも呼ぶ |
| 棚番 | Bin Code / Location Code | ロケーションを識別するコード。例: `A-03-02`（エリアA、棚03、段02） |
| ゾーン | Zone | 倉庫内の区画。温度帯・商品カテゴリ・作業種別で分ける |
| 固定ロケーション | Fixed Location | 商品ごとに決まった保管場所を割り当てる方式 |
| フリーロケーション | Free Location | 空いている場所に自由に格納する方式。スペース効率が良い |
| ダブルトランザクション | Double Transaction | フリーロケーションで、入庫と出庫を別のロケーションで行う方式 |
| ロケーション移動 | Relocation / Transfer | 商品を別のロケーションに移す作業 |
| 間口（まぐち） | Face / Slot | 棚の1区画。商品を手前から取り出せる単位 |
| バッファエリア | Buffer Area | 一時保管場所。入荷後の仮置き場、出荷前の集荷場所など |
| ステージング | Staging | 出荷前の商品を一時的に集めるエリアまたはその作業 |

## 4. 出荷・ピッキング

| 用語 | 英語 | 説明 |
|------|------|------|
| ピッキング | Picking | 出荷指示に基づいて棚から商品を取り出す作業 |
| シングルピッキング（摘み取り） | Discrete Picking | 1注文ずつ棚を回って商品を集める方式 |
| トータルピッキング（種まき） | Batch Picking / Sort-after-pick | 複数注文分をまとめて商品を集め、後から注文ごとに仕分ける方式 |
| マルチオーダーピッキング | Multi-order Picking | 1回の棚回りで複数注文分を同時に集める方式 |
| ピッキングリスト | Pick List | ピッキング作業に必要な商品・数量・ロケーションの一覧 |
| 出荷指示 | Shipping Order | 何をどこにいくつ出荷するかの指示データ |
| 照合 | Verification | 出荷する商品が指示と一致するか確認すること |
| 梱包 | Packing | 商品を箱や袋に詰める作業 |
| 荷合わせ | Consolidation | 同一出荷先の荷物を1つにまとめること |
| 仕分け | Sorting | 出荷先・配送ルートごとに荷物を分類する作業 |
| ウェーブ | Wave | ピッキング作業をグループ化して実行する単位。時間帯や配送先で区切る |

## 5. 在庫評価・出庫ルール

| 用語 | 英語 | 説明 |
|------|------|------|
| 先入先出（FIFO） | First In, First Out | 先に入荷したものから先に出庫するルール。食品・医薬品で必須 |
| 後入先出（LIFO） | Last In, First Out | 後に入荷したものから先に出庫するルール。一般的に倉庫では非推奨 |
| 先期限先出（FEFO） | First Expired, First Out | 有効期限が早いものから先に出庫するルール。FIFO より厳密な期限管理 |
| ロット指定出庫 | Lot-designated Issue | 特定のロットを指定して出庫すること |
| 移動平均法 | Moving Average Method | 在庫の評価額を入庫のたびに平均して算出する方法 |
| 総平均法 | Weighted Average Method | 一定期間の入庫金額を平均して在庫評価する方法 |
| 最終仕入原価法 | Last Purchase Price Method | 直近の仕入価格で全在庫を評価する方法 |

### FIFO/FEFO の実装上の注意

```
FIFOの場合:
  同一SKUの在庫が複数ロットある
  → 入荷日が最も古いロットから引当・出庫する
  → ロケーション内でもロット単位で入荷日を管理する必要がある

FEFOの場合:
  同一SKUの在庫が複数ロットある
  → 有効期限が最も近いロットから引当・出庫する
  → 出荷先によって「残り賞味期限◯日以上」の制約がある場合も
```

## 6. 棚卸し

| 用語 | 英語 | 説明 |
|------|------|------|
| 棚卸し | Stocktaking / Physical Count | 実際の在庫数を数えて理論在庫と照合する作業 |
| 一斉棚卸し | Full Inventory Count | 全在庫を一度に数える。営業を止めて実施することが多い |
| 循環棚卸し（サイクルカウント） | Cycle Count | エリアやカテゴリを区切って日常的に少しずつ数える方式 |
| 差異調整（在庫調整） | Inventory Adjustment | 棚卸し結果に基づいて理論在庫を修正すること |
| 棚卸し差異率 | Inventory Accuracy Rate | 理論在庫と実在庫の一致率。`(一致数 / 総数) × 100` |
| 期首棚卸し | Beginning Inventory | 会計期間の開始時点の在庫 |
| 期末棚卸し | Ending Inventory | 会計期間の終了時点の在庫 |

## 7. ロット・トレーサビリティ

| 用語 | 英語 | 説明 |
|------|------|------|
| ロット | Lot | 同一条件で製造・入荷された商品のまとまり |
| ロット番号 | Lot Number | ロットを識別する番号。入荷日・製造日・仕入先と紐づく |
| シリアル番号 | Serial Number | 個品を一意に識別する番号。1品1番号 |
| 有効期限（使用期限） | Expiry Date / Use-by Date | その日までに使用すべき期限 |
| 賞味期限 | Best Before Date | 品質が保証される期限。期限後も使用可能な場合がある |
| 製造日 | Manufacturing Date | 商品が製造された日付 |
| トレーサビリティ | Traceability | 商品の入荷から出荷までの履歴を追跡できること |
| 正追跡（トレースフォワード） | Trace Forward | 原材料→製品→出荷先の方向に追跡。リコール時に使う |
| 逆追跡（トレースバック） | Trace Back | 製品→原材料→仕入先の方向に追跡。原因調査時に使う |
| リコール | Recall | 出荷済み製品を回収すること。ロット単位で対象を特定する |

## 8. バーコード・ラベル

| 用語 | 英語 | 説明 |
|------|------|------|
| JANコード（EAN） | JAN / EAN | 日本の商品識別コード。13桁（標準）or 8桁（短縮） |
| ITFコード | ITF-14 | ケース（箱）単位の識別コード。14桁 |
| GS1-128 | GS1-128 | 物流標準バーコード。AI（識別子）で商品コード・ロット・期限等を1本に含む |
| GS1データバー | GS1 DataBar | 小さい商品向けのバーコード。生鮮食品等で使用 |
| QRコード | QR Code | 2次元コード。大量の情報を格納できる |
| AI（アプリケーション識別子） | Application Identifier | GS1-128で情報の種類を示す接頭辞。例: `(01)` = GTIN、`(10)` = ロット |
| GTIN | Global Trade Item Number | 国際的な商品識別番号の総称。JANコード/EANコード/UPCを包含 |
| SSCC | Serial Shipping Container Code | パレット・コンテナ単位の識別コード。18桁 |
| SCMラベル | SCM Label | 物流ラベル。SSCC + 出荷元/先 + 商品情報を含む |
| ロケーションラベル | Location Label | 棚に貼付するバーコードラベル。棚番をコード化したもの |

## 9. システム・データ

| 用語 | 英語 | 説明 |
|------|------|------|
| マスタ | Master Data | 商品・ロケーション・取引先など基礎的な情報 |
| 商品マスタ | Item Master | SKUごとの商品名・サイズ・重量・バーコード等の情報 |
| ロケーションマスタ | Location Master | 倉庫内の全ロケーションとそのゾーン・容量等の情報 |
| 取引先マスタ | Business Partner Master | 仕入先・出荷先の情報 |
| トランザクション | Transaction | 入出庫・移動・調整など在庫を変動させる1件の処理 |
| 在庫テーブル | Inventory Table | 現在の在庫状態を保持するデータ。SKU×ロケーション×ロットの組み合わせ |
| 入出庫履歴 | Transaction History | すべての在庫変動の記録。監査証跡として重要 |
| ERP | Enterprise Resource Planning | 基幹業務システム。WMSはERPの在庫管理モジュールと連携することが多い |
| EDI | Electronic Data Interchange | 企業間の電子データ交換。受注・発注・ASNなどを電子的にやり取り |
| API連携 | API Integration | WMSと他システム（ERP・EC・配送）をAPIで接続すること |
| オフライン対応 | Offline Support | 通信断でもHTが動作する仕組み。ローカルに一時保存し、復旧後に同期 |
| 排他制御 | Exclusive Control | 同一在庫を複数人が同時操作した際のデータ矛盾を防ぐ仕組み |

## 11. KPI・指標

倉庫運営の改善に使われる主要な指標。

| 指標 | 英語 | 計算式 / 説明 |
|------|------|--------------|
| 在庫回転率 | Inventory Turnover | `出庫金額 ÷ 平均在庫金額`。高いほど在庫が効率的に回転している |
| 在庫回転日数 | Days of Inventory | `365 ÷ 在庫回転率`。在庫が何日分あるかを示す |
| 棚卸し差異率 | Inventory Accuracy | `(一致件数 ÷ 総件数) × 100`。99.5%以上が目標 |
| 誤出荷率 | Shipping Error Rate | `誤出荷件数 ÷ 総出荷件数`。0.01%以下が目標 |
| ピッキング生産性 | Picking Productivity | `ピッキング行数 ÷ 作業時間`。1時間あたりの処理行数 |
| 入荷リードタイム | Receiving Lead Time | 入荷から入庫完了までの所要時間 |
| 出荷リードタイム | Shipping Lead Time | 出荷指示から出荷完了までの所要時間 |
| 保管効率 | Storage Utilization | `使用ロケーション数 ÷ 総ロケーション数`。スペースの有効活用度 |
| 欠品率 | Stockout Rate | `欠品発生件数 ÷ 総注文件数`。低いほど良い |
| 返品率 | Return Rate | `返品件数 ÷ 総出荷件数`。品質や出荷精度の指標 |
